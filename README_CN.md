## 核心功能
英雄升级时自动获得随机宝物。宝物按等级区间分池抽取，且仅允许白名单内的编号命中，确保奖励可控并兼容各类模组。

## 重制要点（>90% 内容重构）
1. 规则重构：由“黑名单排除”改为“白名单准入”，仅允许指定编号的宝物被抽取，避免无效或冲突的发放。
2. 奖池扩展：新增 `FU(grand_master_lv_up_bonus)`，共有 5 个梯度奖池（`FU(basic_lv_up_bonus)` → `FU(grand_master_lv_up_bonus)`），覆盖从新手到终局的成长阶段。
3. 格式优化：合并连续的编号区间，并为每段添加完整注释（宝物名称与来源模组）。
4. 经验/等级映射调整：Lv1–9 → `FU(basic_lv_up_bonus)`，Lv10–19 → `FU(advanced_lv_up_bonus)`，Lv20–28 → `FU(expert_lv_up_bonus)`，Lv29–39 → `FU(master_lv_up_bonus)`，Lv40+ → `FU(grand_master_lv_up_bonus)`。
5. 兼容性增强：兼容 ERA 3.0+、WoG 3.58f+、进阶职业 Mod v1.094a、三阶升级 Mod Reborn v1.7.3。

## 触发条件
启用 WoG 地图规则 15（英雄升级触发自定义脚本）即可。脚本在英雄升级后自动触发：人类玩家会收到弹窗提示，AI 则静默获得宝物。

## 宝物池设计原则（限制与例外）
### 禁止类（不纳入奖池）
- 改变冒险地图魔法行为的宝物：土书、水书、气书、魔法帽（避免触发跳/飞/回等影响平衡的魔法）。
- 增强战场魔法的宝物：末日之刃、预言者之靴、术士之戒、法师长袍、法师之杖、元素指环等（防止战场魔法强度失衡）。
- 改变地图行动方式的宝物：水鞋、翅膀、海洋帽（避免改变地图探索节奏）。
- 作为合成组件的物品：多数盾牌、组合宝物的顶级组件、气球、土球、火球、泰坦组件（避免通过奖励快速合成强力宝物）。
- 直接奖励组合宝物：防止一步到位获得顶级组合装，保持养成节奏。
- 能新增或解除战场限制的宝物：战争枷锁、禁魔球、沉默披风、毁灭之球、各系免疫类等（保持战场规则稳定）。
- 单件即超模的宝物：职业套装、食人魔铁拳、秘稀宝物等（单件即能显著改变战力）。

### 例外（可纳入奖池）
- 个别带战场魔法但强度可控的宝物：如部分开场施法的弱效果宝物。
- 组合宝物组件例外：金鹅、双筒望远镜、外交官斗篷、丰收号角、森林领主之袍。

## 奖池映射
- `FU(basic_lv_up_bonus)`（Lv1–9）：新手期基础宝物（以普通宝物为主）。
- `FU(advanced_lv_up_bonus)`（Lv10–19）：过渡期低—中阶宝物（普通 + 少量三阶 Mod 宝物）。
- `FU(expert_lv_up_bonus)`（Lv20–28）：发育期中阶宝物（普通 + WoG + 三阶 Mod 宝物）。
- `FU(master_lv_up_bonus)`（Lv29–39）：成型期高—中阶宝物（高阶普通 + 指挥官 + 三阶 Mod 宝物）。
- `FU(grand_master_lv_up_bonus)`（Lv40+）：终局期高阶宝物（顶级普通 + 各模组稀有宝物）。

## 注意事项
1. 宝物发放后将自动装备到英雄身上，请确保英雄有相应空位。
2. 随机 ID 范围为 7–289；若抽到的 ID 不在白名单内，会继续随机直至命中白名单。
3. 重试上限为 100 次；若仍未命中白名单，则发放保底宝物「厄运沙漏」。
